Local Business Website

Overview

This project is a responsive local business website developed using HTML, CSS, and JavaScript. It showcases products with an intuitive user interface and demonstrates fundamental front-end web development concepts.

Features

* Responsive and user-friendly design
* Product listing page
* Shopping cart interface
* Login page
* Clean navigation between multiple pages
* Image-based product display

Technologies Used

* HTML5
* CSS3
* JavaScript

Project Structure

* `index.html` – Home page
* `products.html` – Product catalog
* `cart.html` – Shopping cart
* `login.html` – Login page
* `style.css` – Styling
* `script.js` – Client-side functionality
* `images/` – Product images and assets

How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Navigate through the website using the provided links.

Purpose

This project was created as part of academic learning to practice front-end web development and build a simple business website with multiple pages and interactive functionality.
